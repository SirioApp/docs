# What Is Backed

Backed is capital formation infrastructure for autonomous organizations on MegaETH.

It provides a structured path to:

- link a raise to an existing agent identity,
- create the project onchain,
- explicitly approve the project for launch,
- accept capital commitments in USDM,
- settle the raise according to contract-defined outcomes,
- manage claims, refunds, and post-close ownership flows.

## In plain terms

Backed is designed to help a market participant answer three questions clearly:

1. What exactly is being launched?
2. When is it actually live?
3. Which source defines the financial outcome?

The platform answers those questions by separating identity, creation, approval, participation, and settlement into distinct stages.

## Why Backed exists

Most fundraising interfaces collapse critical distinctions:

- identity and issuance are treated as the same event,
- project creation is confused with public launch,
- privileged actions are obscured behind generic admin behavior,
- interface output is treated as if it were the source of truth.

Backed takes the opposite approach. It makes responsibilities explicit and keeps lifecycle truth onchain.

## What Backed is not

Backed is not a generic campaign page builder and it is not a thin frontend over speculative promises. It is a structured onchain raise system with explicit control points and explicit settlement logic.

## One-sentence summary

Backed turns an approved agent identity into an investable, contract-defined raise with explicit launch controls and deterministic settlement paths.
