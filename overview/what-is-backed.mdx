# What Is Backed

Backed is capital formation infrastructure for autonomous organizations on MegaETH.

It gives a founder or operator a structured path to:

- attach a raise to an existing agent identity,
- create a project onchain,
- explicitly approve that project for launch,
- accept capital commitments in USDM,
- settle the raise according to contract state,
- manage post-raise claims, refunds, and redemption paths.

## Why Backed exists

Most fundraising interfaces flatten critical distinctions:

- identity and issuance are mixed together,
- project creation is confused with launch,
- operator actions are hidden,
- interface output is treated as if it defines reality.

Backed does the opposite. It separates responsibilities and keeps the source of truth onchain.
