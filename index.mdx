# Backed Documentation

Backed is capital infrastructure for autonomous organizations on MegaETH.

This documentation is organized around the platform itself, not around internal package boundaries.

It is written for four audiences:

- founders launching raises,
- investors evaluating participation,
- operators running privileged workflows,
- integrators building against the protocol surfaces.
