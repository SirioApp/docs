# Contracts

Backed is built around a small set of contracts with distinct responsibilities.

## Core contracts

### AgentRaiseFactory

Primary control contract for:

- project creation
- project discovery
- project metadata
- approval state
- global configuration

### Sale

Per-project sale contract responsible for:

- commitments
- finalization
- claims
- refunds
- collateral accounting

### AgentVaultToken

The vault share token representing post-raise ownership after settlement.

### ContractAllowlist

Execution guardrail used to restrict treasury interactions with external contracts.

### SafeModuleSetup

Deployment-side setup used when treasury control is routed through Safe modules.

## External contracts

### ERC8004 Identity Registry

The identity prerequisite for raise creation. The factory reads the agent wallet from this registry.

### USDM

The platform collateral token used for sale commitments in the current deployment model.

## Contract-level operating rule

If interface output and contract state disagree, trust the deployed contracts.
